# PDF 智能解析工具 — 使用说明

## 这是什么？

一个能把 PDF 文件中的文字提取出来的工具。
- 如果是**文字型 PDF**（直接能复制的）→ 自动提取文字
- 如果是**扫描型 PDF**（每页是图片的）→ 自动 OCR 识别文字

支持导出为 Markdown、TXT、JSON 格式。

---

## 快速开始

### 方式一：直接双击 EXE（最简单，推荐）

打开 `dist/` 文件夹，双击 **`PDF解析工具.exe`** 即可使用。
> 无需安装 Python，无需配置环境，开箱即用。

### 方式二：运行 Python 脚本（需要 Python 环境）

```bash
# 1. 安装依赖（只需第一次）
pip install pymupdf rapidocr-onnxruntime

# 2. 运行程序
python pdf_parser_desktop.py
```

### 方式三：自己打包 EXE

```bash
# 1. 安装打包工具
pip install pyinstaller

# 2. 运行打包脚本（等待 1-3 分钟）
python pack_for_sharing.py

# 3. 在 dist/ 文件夹找到 PDF解析工具.exe
```

## 界面操作说明

### 主界面分为左右两栏

**左侧 — 操作面板：**

1. **选择文件** → 点击「浏览」选择 PDF 文件
2. **查看文件信息** → 自动显示标题、作者、页数
3. **选择解析方式：**
   - `自动检测`（推荐）→ 有文字直接提取，没文字自动 OCR
   - `文字提取` → 强制只用文字模式
   - `OCR 识别` → 强制用 OCR 识别图片
4. **设置 DPI** → 数字越大 OCR 越清晰但越慢（默认 300）
5. **设置页数** → 0 表示全部，可以只解析前几页测试
6. **点击「开始解析」** → 等待进度条跑完

**右侧 — 预览面板：**
- 解析完成后可以翻页查看每页的文字内容
- 使用 ◀ ▶ 按钮切换页面

**导出按钮：**
- `导出 MD` → Markdown 格式（带标题层级）
- `导出 TXT` → 纯文本格式
- `导出 JSON` → 结构化数据（适合程序处理）

## 常见问题

### Q：OCR 是什么？
OCR（光学字符识别）就是把图片里的文字"读"出来。扫描版 PDF 每页都是一张图片，需要用 OCR 才能变成文字。

### Q：OCR 识别速度怎么样？
- 每页约 2-5 秒（取决于 DPI 设置和电脑配置）
- 300 DPI 是速度和质量的平衡点
- 300 页的 PDF 大概需要 10-20 分钟

### Q：支持英文 PDF 吗？
支持。RapidOCR 支持中英文混合识别。

### Q：识别出来的文字有错别字怎么办？
OCR 不可能 100% 准确，中文扫描件通常有 95% 以上的准确率。如果要求更高，可以：
- 提高 DPI 到 400-600（但速度会变慢）
- 导出后用文本编辑器手动修正
